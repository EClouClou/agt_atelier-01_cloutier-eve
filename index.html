UTILISATION DE & DANS DES STYLES IMBRIQUÉS (CSS NESTING)

Avantages

L'imbrication des styles diminue les doublons, économise de l'espace, aide à la lisibilité et à la maintenance du CSS.

    Ces deux déclarations donnent ce même résultat :

    Vous trouverez ici le lien vers le site de Google.

  • Voici la façon classique de le déclarer : 
    .style-imbrique {
    font-weight: 500;
}
                    
.style-imbrique > a:hover,
.style-imbrique > a:hover {
    color:green;
    color: white;
    font-weight: 900;
    background-color: green;
}
  • Voici la nouvelle méthode qui utilise le " & " sélecteur d'imbrication : 
    .style-imbrique {
    font-weight: 500;
    & > a {
        color:green;
        &:hover,
        &:focus {
            color: white;
            font-weight: 900;
            background-color: green;
        }
    }
}

PSEUDO-CLASSE :HAS

Avantages

Le :has aide à mettre en forme de manière plus précise et conditionnelle au moins un élément qui fait partie d'un groupe d'éléments préalablement ciblés en CSS.

Ici, on s'assure que le h4 a une marge de dessous plus petite s'il a un h5 directement sous lui. Sinon, cette marge a même pour h4 et h5.

h4 {
    color: var(--grey-color);
    font-size: 1rem;
    margin: 0 0 1rem 0;
}

h5 {
    color: var(--black-color);
    font-weight: 400;
    font-size: 0.8rem;
    margin: 0 0 1rem 0;
}

h4:has(+ h5) {
    margin: 0 0 0.25rem 0;
}

TECHNIQUES DE RENVOI À LA LIGNE DU TEXTE (TEXT-WRAP) : BALANCE ET PRETTY

Avantages

Ces fonctionnalités aident à créer des blocs de texte agréables à l'oeil, évitent les orphelins et balancent la répartition des traits d'union. L'effet peut paraître subtil, mais il semble que c'est un bon choix pour s'assurer que nos textes, une fois traduits dans les différents navigateurs, soient équilibrés.

Voici la déclaration CSS pour l'exemple avec ces techniques :

.balance-pretty {
    text-wrap: balance;
    text-wrap: pretty;
}

LETTRINE (INITIAL-LETTER)

Avantages

Cette option imbriquée toute simple, mais très esthétique permet le positionnement et l'adaptation rapide et d'une lettrine en début de phrase à l'aide de deux arguments : le premier marque la profondeur de la lettre dans le paragraphe et le second, la hauteur par rapport aux lignes du paragraphe.

Voici la déclaration du texte qui suit :

.histoire {
    font-family: "Kalnia", serif;
    font-size: 1.5rem;
    font-weight: 200;
    color: var(--red-color);

    &::first-letter {
        initial-letter: 4 3;
        font-weight: 900;
        padding-right: 1rem;
        background: linear-gradient(to right, var(--red-color), var(--orange-color));
        color: transparent;
        -webkit-background-clip: text;
    }
}
